Docker in SCATCH — 30 second explanation
Your computer = a city. Every software you install = a building you construct from scratch on your land.
Docker = shipping containers. Redis comes pre-packed in a container. You just drop it on your land, it runs. Remove it, nothing left behind. No messy installs.
In SCATCH specifically:

Your Express server will talk to Redis at localhost:6379
Redis will cache product listings, rate-limit auth routes, store session data
You just run docker start scatch-redis before coding, done
When you deploy, you switch to Upstash (cloud Redis) — zero code change, just swap the URL in .env

That's it. Docker = clean, portable, disposable software boxes.🔥
